# Voice Task Master (VTM) v1.5.0

A voice-first productivity extension for the Hack Club Flavortown community. Manage project backlogs with speech and a real-time HUD on project pages.

## Features

- Add or manage tasks using natural speech (Web Speech API).
- Real-time HUD injected into `flavortown.hackclub.com`.
- Detects active Flavortown project and tags tasks with context.
- Priority glows (Critical/Ship/Backlog) based on command analysis.
- Live transcription feedback (interim results) while speaking.
- Global voice recognition works even when popup is closed.

## Hotkeys

- `Ctrl+Shift+V` — Start voice recognition from any page.
- `Ctrl+Shift+Y` — Open popup (alternate shortcut).

## Installation

1. Download the latest zip from this repository.
2. Open `chrome://extensions` and enable Developer Mode.
3. Click "Load unpacked" and select the extracted folder.
